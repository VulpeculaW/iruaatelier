# Nebula Staff | 星雲の杖

> 无需 Action Menu，仅凭手部动作即可挥舞的星光魔法杖。✦

[在 Booth 查看](https://vulpeculakitsune.booth.pm/items/6052328)

![Nebula Staff 预览](/nebula-staff.jpg)

## ✨ 这个机关可以做什么

这是一款魔法杖粒子机关，**无需操作 Action Menu**，仅凭手部动作即可完成所有交互。
它基于 Modular Avatar 制作，设置会在 Build 或进入 Play Mode 时自动应用。

关于 Modular Avatar 的详细信息，请参考[官方文档](https://modular-avatar.nadena.dev)。

## 📦 套装构成

所有套装的机关都是相同的。区别仅在于法杖的设计，以及是否包含紫水晶配色。

- **Digital Full Package** — 包含全部设计 + 紫水晶配色
- **Digital Tradition Pack** — 传统设计
- **Digital Modern Pack** — 现代设计

## 🧰 需要准备

- Unity 2022.3.22f1
- VRChat Avatar SDK | ↑ 3.7.0
- Modular Avatar | ↑ 1.9.15
- 套装中附带的 Nebula Staff Prefab

::: warning 运行注意事项
- 本资产使用 **Particle System**。若在 VRChat 图形设置中启用了 `Particle Limit`，粒子可能无法显示。请将其设为 **OFF**。
- 本资产使用 **交互机关**。请在 VRChat 设置中将 `Interaction`（Self Interaction）设为 **ON**。
- 由于 VRChat 规格变更，自己的粒子只有在被镜子或相机投影时才会进行模拟。从第三者视角观看时会正常显示。
:::

## 🛠 安装 / 设置

### 1. 基本导入

把套装中附带的 Nebula Staff Prefab 放到 Avatar Root 下。

请将提供的 Prefab 适当地对齐到你角色的手部和背部位置。

### 2. 应用

没有单独的 Apply 按钮。
在 Build 或进入 Play Mode 时会自动应用。

## 🎮 实际操作方法

所有操作都无需 Action Menu，仅凭手部动作完成。

### 1. 召唤法杖 (Summon Staff)

用手 Grab 背后法杖的 `Core` 部分。

### 2. 解除法杖 (Release Staff)

抓住法杖装饰的末端并向外拉。

### 3. 普通射击 (Normal Shot)

用持杖的手进行 Grab。

### 4. 蓄力射击 (Charging Shot)

把中间的珠子放入法杖的 `Core`，然后将法杖旋转一圈。

### 5. 魔法模式 (Magic Mode)

将法杖快速顺时针旋转（左手为逆时针）。

### 6. 雨伞模式 (Umbrella Mode)

处于魔法模式时，将法杖垂直竖立 1 秒以上。

### 7. 解除雨伞模式 (Turn off Umbrella Mode)

将法杖水平放置 1 秒以上。

## ❓ 常见问题

**Q. 看不到粒子效果。**
A. 请检查 VRChat 图形设置中的 `Particle Limit`。该设置需为 **OFF**，粒子才会正常显示。

**Q. 交互机关不工作。**
A. 请在 VRChat 设置中将 `Interaction` 设为 **ON**。

**Q. 在镜子和相机中粒子显示异常。**
A. 由于 VRChat 规格变更，自己的粒子只有在被镜子或相机投影时才会进行模拟。从第三者视角观看时会正常显示。
