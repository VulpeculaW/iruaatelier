# Nebula Staff | 星雲の杖

> Action Menu 없이, 손동작 하나로 펼치는 별빛 마법의 지팡이. ✦

[Booth에서 보기](https://vulpeculakitsune.booth.pm/items/6052328)

![Nebula Staff 프리뷰](/nebula-staff.jpg)

## ✨ 무엇을 하는 기믹인가요?

이 기믹은 모든 상호작용을 **Action Menu 조작 없이** 손동작만으로 사용하는 마법의 지팡이 파티클입니다.
Modular Avatar 기반으로 제작되었으며, 설정은 Build 또는 Play 시 자동 적용됩니다.

Modular Avatar에 대한 자세한 내용은 [공식 문서](https://modular-avatar.nadena.dev)를 참고하세요.

## 📦 패키지 구성

모든 패키지의 기믹은 동일합니다. 지팡이의 디자인과 자수정 색상 포함 여부에만 차이가 있습니다.

- **Digital Full Package** — 모든 디자인 + 자수정 색상 포함
- **Digital Tradition Pack** — 전통 디자인
- **Digital Modern Pack** — 모던 디자인

## 🧰 준비물

- Unity 2022.3.22f1
- VRChat Avatar SDK | ↑ 3.7.0
- Modular Avatar | ↑ 1.9.15
- 패키지에 포함된 Nebula Staff Prefab

::: warning 동작 주의사항
- 이 에셋은 **Particle System**을 사용합니다. VRChat 그래픽 설정에서 `Particle Limit`가 켜져 있으면 파티클이 보이지 않을 수 있습니다. **OFF**로 설정해주세요.
- 이 에셋은 **상호작용 기믹**을 사용합니다. VRChat 설정에서 `Interaction`(Self Interaction)을 **ON**으로 켜주세요.
- VRChat 사양 변경으로 자기 자신의 파티클은 거울이나 카메라에 투영되고 있을 때만 시뮬레이션됩니다. 제3자의 시점에서는 정상적으로 표시됩니다.
:::

## 🛠 설치 / 설정

### 1. 기본 도입

아바타 Root에 패키지에 포함된 Nebula Staff Prefab을 넣어주세요.

제공된 프리펝을 아바타의 손과 등의 위치에 적절하게 맞춰주세요.

### 2. 적용

별도의 Apply 버튼은 없습니다.
Build 하거나 Play Mode에 들어가면 자동 적용됩니다.

## 🎮 실제 조작 방법

모든 조작은 Action Menu 없이 손동작으로 수행합니다.

### 1. 지팡이 소환 (Summon Staff)

등에 있는 지팡이의 `Core` 부분을 손으로 Grab 합니다.

### 2. 지팡이 해제 (Release Staff)

지팡이의 장식 끝부분을 잡고 당깁니다.

### 3. 노말 샷 (Normal Shot)

지팡이를 가지고 있는 손으로 Grab 합니다.

### 4. 차징 샷 (Charging Shot)

중간에 있는 구슬을 지팡이의 `Core`에 넣은 후, 지팡이를 한 바퀴 돌립니다.

### 5. 마법 모드 (Magic Mode)

지팡이를 빠르게 시계 방향(왼손은 반시계 방향)으로 돌립니다.

### 6. 우산 모드 (Umbrella Mode)

마법 모드일 때 지팡이를 수직으로 1초 이상 세웁니다.

### 7. 우산 모드 해제 (Turn off Umbrella Mode)

지팡이를 수평으로 1초 이상 눕힙니다.

## ❓ 자주 묻는 질문

**Q. 파티클이 보이지 않습니다.**
A. VRChat 그래픽 설정에서 `Particle Limit`를 확인해주세요. 해당 설정이 **OFF**여야 파티클이 정상적으로 표시됩니다.

**Q. 상호작용 기믹이 작동하지 않습니다.**
A. VRChat 설정에서 `Interaction`을 **ON**으로 바꿔주세요.

**Q. 거울과 카메라에서 파티클이 이상하게 표시됩니다.**
A. VRChat의 사양 변경으로 자기 자신의 파티클은 거울과 카메라에 투영되고 있을 때만 시뮬레이션됩니다. 제3자의 시점에서는 정상적으로 표시됩니다.
